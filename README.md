# Rust Leptos Playground 

## Setup

1. Install the Rust's WASM target.
    ```shell
    rustup target install wasm32-unknown-unknown
    ```

2. Install Trunk:
    ```shell
    cargo install trunk
    ```

## Serve web-app

Run the following command from the repositories root:

```shell
trunk serve --release --open
```
